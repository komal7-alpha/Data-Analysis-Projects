# Data Analysis Roadmap – 2026

This repository contains a structured **60-day roadmap** to build strong, job-ready skills as a **Data Analyst in 2026**.

The roadmap is designed to take a learner from foundational concepts to advanced analytics, real-world projects, and portfolio preparation.

---

## Overview

This is a complete **2-month learning plan** to become a skilled Data Analyst, covering:

- SQL
- Excel
- Python
- Pandas & NumPy
- Statistics
- Data Visualization
- Power BI
- Real-world projects and portfolio building

The focus is on practical learning, analytical thinking, and hands-on project work.

## Month 1 – Foundations (Days 1–30)

**Goal:** Build a strong base in data handling, querying, and analysis fundamentals.

This phase focuses on understanding data, working with structured datasets, and learning the core tools used by data analysts.

---

### Week 1 (Days 1–7): Excel for Data Analysis

1. Excel interface, formulas, data cleaning  
2. Conditional formatting, sorting, filtering  
3. Pivot tables and charts  
4. Advanced formulas (VLOOKUP, INDEX-MATCH)  
5. Data validation and text functions  
6. Dashboard creation basics  
7. Mini Excel project – Sales report dashboard  

---

### Week 2 (Days 8–14): SQL Foundations

1. SQL basics: SELECT, FROM, WHERE  
2. ORDER BY, DISTINCT, LIMIT  
3. Aggregations: GROUP BY, HAVING  
4. Joins: INNER, LEFT, RIGHT, FULL  
5. Subqueries and CTEs  
6. Window functions (RANK, ROW_NUMBER)  
7. SQL mini-project – employee or sales database  

---

### Week 3 (Days 15–21): Python for Data Analysis

1. Python setup and Jupyter Notebook basics  
2. Data types, loops, and conditionals  
3. NumPy basics – arrays and operations  
4. Pandas: reading CSV files and DataFrames  
5. Data cleaning (missing values, duplicates)  
6. Pandas grouping, merging, and aggregation  
7. Mini Python project – analyze a small dataset  

---

### Week 4 (Days 22–30): Data Visualization

1. Visualization principles and best practices  
2. Matplotlib basics (bar, line, pie charts)  
3. Seaborn and Plotly for interactive visuals  
4. Storytelling with data  
5. Power BI setup and interface  
6. Building Power BI reports  
7. Mini Power BI project – Sales or HR dashboard  

---

## Month 2 – Advanced Concepts & Projects (Days 31–60)

**Goal:** Apply analytical skills to real-world problems and prepare a professional portfolio.

This phase focuses on advanced SQL, deeper analysis, business metrics, dashboards, and job readiness.

---

### Week 5 (Days 31–37): Statistics & Analytics

1. Descriptive statistics (mean, median, mode, standard deviation)  
2. Probability fundamentals  
3. Correlation and covariance  
4. Hypothesis testing and p-values  
5. A/B testing fundamentals  
6. Mini project – Customer churn analysis  

---

### Week 6 (Days 38–44): Advanced SQL & Python

1. Complex joins and CASE statements  
2. Stored procedures and functions  
3. SQL optimization and indexing  
4. Advanced Pandas (apply, map, lambda, joins)  
5. Data transformation and feature engineering  
6. Mini project – Exploratory Data Analysis (EDA)  

---

### Week 7 (Days 45–51): Power BI & Dashboards

1. Power Query transformations  
2. DAX formulas and measures  
3. Building multi-page dashboards  
4. Data storytelling and KPI metrics  
5. Power BI project – Sales or Marketing dashboard  

---

### Week 8 (Days 52–60): Portfolio & Final Project

1. Capstone project – complete data workflow  
2. Resume and GitHub portfolio preparation  
3. LinkedIn portfolio post and Power BI publish  
4. Mock interviews and case study practice  
5. Final review – SQL, Python, and Power BI  

---

## Outcome

By the end of this roadmap, the learner will have:
- Strong fundamentals in data analysis
- Hands-on project experience
- A structured GitHub portfolio
- Job readiness for Data Analyst roles
